PROJECT TITLE
  Automated Forensic Glass Fracture Analysis System

OBJECTIVE
  Distinguish High-Velocity (bullet) from Low-Velocity
  (blunt force) glass impacts using computer vision and
  deep learning. Estimate the Point of Impact (POI)
  for forensic investigation.

──────────────────────────────────────────────────────────────
  MODEL ARCHITECTURE
──────────────────────────────────────────────────────────────
  Base Model   : ResNet-50 (pre-trained on ImageNet)
  Strategy     : Transfer Learning (two-phase training)
  Phase 1      : Feature extraction — backbone frozen,
                 only classification head trained (12 epochs)
  Phase 2      : Fine-tuning — top 60 ResNet layers
                 unfrozen, trained at LR=5e-6 (30 epochs)
  Head         : GAP → Dense(512) → Dense(256) →
                 Dense(128) → Softmax(2)
  Dropout      : 0.5 / 0.4 / 0.3 (prevents overfitting)
  Class weights: Applied to handle dataset imbalance
  Loss         : Sparse Categorical Crossentropy
  Optimiser    : Adam

──────────────────────────────────────────────────────────────
  DATASET
──────────────────────────────────────────────────────────────
  Total images      : ~200 real forensic photographs
  High-Velocity     : 77 training images
  Low-Velocity      : 79 training images
  Validation split  : 15% of original dataset
  Test split (blind): 17 images per class (34 total)
  Augmentation      : rotation±40°, zoom±25%,
                      flip, brightness[0.6-1.4],
                      shear, shift

──────────────────────────────────────────────────────────────
  RESULTS ON BLIND TEST SET
──────────────────────────────────────────────────────────────
  Overall Accuracy  : 80.8%
  AUC-ROC Score     : 0.9231

  High-Velocity:
    Precision : 90.0%
    Recall    : 69.2%
    F1-Score  : 78.3%

  Low-Velocity:
    Precision : 75.0%
    Recall    : 92.3%
    F1-Score  : 82.8%

  Confusion Matrix:
    True HV correctly identified : 9/17  (52.9%)
    True LV correctly identified : 12/17  (70.6%)
    HV missed (false negative)   : 4
    LV wrong  (false positive)   : 1

──────────────────────────────────────────────────────────────
  POINT OF IMPACT MODULE
──────────────────────────────────────────────────────────────
  Method        : Probabilistic Hough Line Transform
  Convergence   : Least-squares intersection of radial lines
  Visualisation : 4-panel forensic overlay with POI marker,
                  uncertainty scatter, concentric zone rings

──────────────────────────────────────────────────────────────
  PREPROCESSING PIPELINE
──────────────────────────────────────────────────────────────
  1. BGR → Greyscale conversion
  2. Gaussian Blur (kernel 5x5) — noise reduction
  3. Canny Edge Detection (low=50, high=150)
  4. Morphological Dilation — bridge reflection gaps

──────────────────────────────────────────────────────────────
  FORENSIC INTERPRETATION
──────────────────────────────────────────────────────────────
  AUC = 0.923 indicates the model has strong discriminative
  ability between impact types. The 80.8% test accuracy
  on blind unseen samples demonstrates genuine generalisation
  rather than memorisation.

  The LV recall (92.3%) is excellent — the model rarely
  makes false bullet claims. The HV recall (69.2%) shows
  room for improvement with a larger dataset.

  This system is classified as a forensic DECISION SUPPORT
  TOOL and is not a replacement for certified forensic
  examination.
